# 06 黑白棋玩法規格

*更新：2026-07-12（v0.3）*

## 一、規則與操作

- 8×8 標準黑白棋（Othello），玩家執黑先手，AI 執白。
- **合法手提示**：玩家回合時，可落子的格子顯示半透明小圓點。
- 落子自動翻轉被夾的棋子；最後一手有橙色標記；比分（黑 n：n 白）即時顯示。
- **跳過回合**：一方無合法手時彈出提示對話框，由另一方繼續；雙方皆無合法手即終局。
- **悔棋**：回到玩家上一手落子前的盤面（快照還原，一次收回雙方各一手）。
- **自動存檔**：存整個盤面 + 輪到誰（64 格資料量小），可從首頁「繼續」還原。
- 終局以子數多者勝，記入 `reversi_stats`（場次、勝場、各難度勝場）。

## 二、AI 設計（`ReversiLogic`）

重用五子棋的 Negamax + Alpha-Beta 框架，替換走子規則與評估函數：

| 難度 | 策略 | 實測速度 |
| --- | --- | --- |
| 初學 ★ | 隨機合法手 | 即時 |
| 普通 ★★ | 深度 1（貪婪 + 位置權重） | 即時 |
| 困難 ★★★ | 深度 3 | 即時 |
| 專家 ★★★★ | 深度 4；剩 ≤8 空格時算到終局（精確） | ~340 ms/手 |

評估函數（分階段）：

- **中盤**：經典位置權重表（角 +100、角旁 X 位 −50）＋ 行動力差（合法手數量）× 8
- **終盤（≤10 空格）**：改看子數差
- 搜尋時以位置權重排序候選手，加速 Alpha-Beta 剪枝

實測強度分層：困難 vs 普通完整對局 59:4。
