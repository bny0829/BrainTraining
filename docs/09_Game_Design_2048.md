# 09 2048 玩法規格

*更新：2026-07-12（v0.6）*

## 一、規則與操作

- 4×4 盤面，**滑動**（觸控手勢）或**方向鍵**（桌面）移動全部磚塊，相同數字相撞合併。
- 一次滑動中每個磚塊最多合併一次（[2,2,2,2] → [4,4]，不會連鎖成 8）。
- 每次有效移動後在隨機空格生成新磚：90% 出 2、10% 出 4。
- 合併得分 = 合成磚塊的數值，累計為本局分數。
- **達成 2048** 彈出祝賀，可繼續挑戰更大數字；無法移動時結束。
- **復原**：單步（防手滑），無次數限制設計上僅保留最近一步。
- **自動存檔**：每步即存，可從首頁「繼續」。

## 二、紀錄與成就

- `game2048_stats`：最高分（best_score）、最大磚塊（best_tile）、完整場數（played）——最佳紀錄即時更新，不用打完整場。
- 成就：小有所成（合成 512）、2048！（合成 2048）。
- 無難度分級（此遊戲深度來自策略而非參數）。

## 三、設計備註

- 磚塊配色沿用 2048 經典色系（淺米 → 橙 → 金黃），超過 2048 的磚塊統一深色。
- 滑動判定：位移 ≥ 40px、取水平/垂直分量較大者。
- 未加入每日挑戰輪替：單局時間過長（10 分鐘以上）不適合每日節奏；未來可做「N 步內合成 256」限步模式再納入。
